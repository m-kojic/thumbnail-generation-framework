# Thumbnail Generation Framework: AWS CDK Infrastructure

> **About this project:**
> This component is part of the **Thumbnail Generation Framework**, a reusable C# framework designed for processing video thumbnails using Clean Architecture and Hexagonal design principles. The full pipeline supports local and distributed (AWS) deployments. For a comprehensive overview, see the [root README.md](../../README.md).

---

## Welcome to the CDK C# project!

This is a blank project for CDK development with C#.

The `cdk.json` file tells the CDK Toolkit how to execute the app.

It uses the [.NET CLI](https://docs.microsoft.com/dotnet/articles/core/) to compile and execute the project.

## Useful commands

* `dotnet build src` compile this app
* `cdk deploy`       deploy this stack to your default AWS account/region
* `cdk diff`         compare deployed stack with current state
* `cdk synth`        emits the synthesized CloudFormation template
